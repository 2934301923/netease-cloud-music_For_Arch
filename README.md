#### 介绍
用于解决Arch系列系统网易云无法输入中文的问题<br>
本人Manjaro+KDE测试成功

#### 安装教程
由于本人已把qcef及网易云放到AUR，亦可通过AUR安装
首先安装qcef

```
yay -S qcef
```

再安装网易云

```
yay -S netease-cloud-music-for-arch
```


安装完毕！
